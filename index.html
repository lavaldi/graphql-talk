Problemáticas que resuelve GraphQL

  1. Dificultad en decidir la estructura de la URL cuando se tienen relaciones muy anidadas.
  2. Podemos ejecutar fácilmente multiples solicitudes http para obtener la data que necesitamos.
  3. Sobrecarga de datos.

¿Qué es GraphQL?

GraphQL es un lenguaje de consulta para tu API, y un runtime del lado del servidor para ejecutar consultas mediante el uso de un sistema de tipado (schema) que defina para tus datos.
